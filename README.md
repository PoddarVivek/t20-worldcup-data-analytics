# 🏏 T20 World Cup Data Analytics

This project analyzes ICC Men's T20 World Cup data to uncover player performance insights, team statistics, and match trends.

## Tools Used
- Python
- Pandas
- Power BI
- Data Visualization

## Project Workflow
1. Data Collection
2. Data Cleaning (Python)
3. Data Analysis
4. Dashboard Creation (Power BI)
